# Portfolio

A personal portfolio website built with Vite, React, TypeScript, Tailwind CSS, and Radix UI.

## Overview

This project showcases a developer portfolio with sections for:
- Hero
- About
- Skills
- Projects
- Education
- Certificates
- Leadership
- Contact

It includes a modern responsive layout, interactive UI components, and client-side routing.

## Technologies

- Vite
- React
- TypeScript
- Tailwind CSS
- Radix UI / shadcn-style components
- React Router
- React Query

## Run locally

```bash
npm install
npm run dev
```

Then open the local development URL shown in the terminal.

## Build

```bash
npm run build
```

## Notes

The app is organized under `src/`, with reusable components in `src/components/` and page routes in `src/pages/`.
